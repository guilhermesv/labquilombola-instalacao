# O repositório

Este repositório abriga os códigos e documenta o desenvolvimento da instalação que desenvolvi para a primeira residência [Lab Quilombola](https://maranha.com.br/labquilombola/).

# A residência Lab Quilombola

A residência Lab Quilombola tem por objetivo aproximação entre mestras e mestras da cultura afro-brasileira com artistas e ativistas da cultural digital. Mais informações sobre a residência podem ser lidas no site: [https://maranha.com.br/labquilombola/](https://maranha.com.br/labquilombola/).
